# 🍔 Swiggy Clone Project

A responsive **Swiggy Clone Web Application** built to simulate the basic functionality and user interface of the Swiggy food delivery platform. This project demonstrates frontend development skills and web application structure using modern technologies.

---

## 🚀 Features

* 🏠 Home page with restaurant listings
* 🔍 Search functionality for food/restaurants
* 📋 Menu display interface
* 🛒 Add to cart functionality
* 📱 Responsive design for mobile & desktop
* 🎨 Clean and modern UI inspired by Swiggy
